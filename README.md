# Yandex Tracker for Obsidian 
Плагин для создания задач в Яндекс.Трекер прямо из Obsidian.

# Как использовать

1. Включите плагин
2. Укажите настройки

## Создание задач

Введите `@ABC` и нажмите "Пробел", где `ABC` - ключ очереди задач.
Откроется окно создания задачи.
После создания задачи ссылка будет автоматически вставлена в тексте.

## Ссылка на задачу

Введите `@ABC-123` и нажмите "Пробел", где `ABC-123` - номер задачи.
Ссылка на задачу будет автоматически вставлена в тексте.
